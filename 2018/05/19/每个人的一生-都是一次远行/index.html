每个人的一生 都是一次远行

发表于 2018-05-19 | 分类于 精神家园 | | 阅读次数:

尹吾，第一次听说这个名字是在2017年上半年，当时听了他的那首《或许》，印象很深。从歌曲里能感受到他内心积蓄的强大力量，借着轻摇滚的方式，渐渐表达出来，这种风格让我联想到了Beyond的《真的爱你》，《喜欢你》等歌曲，虽然他们表达的主题不同。

沉浸在这首歌里的同时，不禁也会问为什么这么好听的歌曲（至少从个人角度说）没有广泛流传？知乎上有一些关于尹吾的介绍，看完之后，他的形象在脑海中也变的清晰了：理想，现实，矛盾，彷徨，挣扎，以及温馨和幸福。这是他的经历，可能也是每个人都要经历的。在此，我也想把他的歌推荐给各位，希望你能够喜欢。

这里我摘抄了部分《或许》的歌词如下：

  • 一个人要把肉身放在岁月的砧板上煅打多少次，他的心才能坚冷如钢
  • 一个人要让泪水浸泡过多少次，那他的眼神才不会迷惘
  • 一条路要走多长，才能抵达远方
  • 一首歌要唱多久，人们才不会遗忘
  • 一条河要绕过多少高山，多少峡谷，才能看见海洋
  • 一朵花该怎样绽放，才能给所有善良和负重的人们，送去安慰和生活的芬芳
  • 或许我们追求了一生，仍要从追求本身寻找
  • 或许答案不在远方，而在你我的心上
